# Algorithm Paper Archive

This archive records candidate algorithm and supply-chain reference metadata
used to design RepoGrammar v0.1. It is metadata-only until source selection,
license terms, and redistribution terms are checked for each entry.

Each domain directory contains:

- `manifest.json`: source metadata, archive status, milestone use, and
  implementation notes.
- `README.md`: a short summary of how the source family constrains
  implementation.

Do not commit downloaded PDFs, HTML snapshots, or other third-party artifacts
until their license permits repository redistribution. When an artifact is kept
locally but not committed, record its path and SHA-256 in the relevant manifest
before relying on it for implementation.
